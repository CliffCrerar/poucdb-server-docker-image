# PouchDB Docker Image

## Prerequisites
1. Nodejs & Npm
2. Docker

---

#### Install dependancies

`npm i`

#### Build image

`npm run build`

#### Run image

`npm run image`

---

#### Test run

For test running the server before building a container:

`npm start`
